# Comunidad Nexo website concept

Landing inicial para Comunidad Nexo.

## Enfoque

- Sitio estático, rápido y fácil de publicar en GitHub Pages.
- Diseño moderno, cálido y editorial.
- Inspirado en patrones buenos de sitios de iglesias contemporáneas:
  - mensaje principal claro,
  - ruta para visitantes nuevos,
  - enseñanzas/podcast,
  - comunidad,
  - eventos,
  - contacto.

## Personalización pendiente

Actualizar cuando la iglesia confirme:

- Horario de reunión.
- Dirección exacta.
- WhatsApp/correo oficial.
- Nombres de liderazgo si se desean mostrar.
- Información de niños, jóvenes, grupos y ofrendas.
- Fotografías reales de la comunidad.

## Desarrollo local

Abrir `index.html` directamente o servir con:

```bash
python3 -m http.server 8080
```
